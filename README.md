# Bakkes Mod Linux Files
Files that are used in the bakkes mod install on linux tutorial

This assumes you have steam installed as a system package. It might work on the flatpack version but I have not tested it.

# bakkers.sh Usage
Ensure that all the paths in the file correspond to things that exist on your system.

This is a folder
/home/fulmetal/.steam/steam/steamapps/compatdata/252950/pfx

Thease 2 are files 
/home/fulmetal/.steam/steam/steamapps/common/Proton - Experimental/files/bin/wine64 
/home/fulmetal/.steam/steam/steamapps/compatdata/252950/pfx/drive_c/Program Files/BakkesMod/BakkesMod.exe

If you are running a different proton version, you must change the second path to the wine64 file that you are using

As an example for proton 8
/home/fulmetal/.steam/steam/steamapps/common/Proton 8.0/dist/bin/wine64
